Plugin jQuery

jQuery.zoomOver

Instalação
Install

Para a instalação são necessários os seguintes ficheiros:

  • jQuery versão 1.4.4 ou superior (poderá funcionar com versões inferiores)
  • jquery.zoomOver.js versão 0.0.1 ou superior
  • zoomOver.css
  • Exemplo completo

Inserir as referências no head

<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
<title>jQuery - zoomOver</title>

<link rel="stylesheet" href="css/style.css" type="text/css" media="all" />
 <link rel="stylesheet" href="css/zoomOver.css" type="text/css" media="all" />

<script type="text/javascript" src="js/jquery-1.4.4.min.js"></script>
<script type="text/javascript" src="js/jquery.zoomOver.js"></script>
